BBC Compilation Test

This test will open the 8bitworkshop IDE and test BBC compilation.

Test Steps:

  1. Open the IDE: http://localhost:8080/
  2. Select "BBC Micro" from the platform dropdown
  3. Try to compile this code:
#include <stdio.h>

int main(void) {
    puts("Hello World!");
    return 0;
}

Expected Result:

If the compilation works, you should see:

  • ✅ No compilation errors
  • ✅ A compiled binary (around 1796 bytes)
  • ✅ The program should run in the BBC emulator

If it fails:

Please share the exact error message you see in the compilation output.

Current Configuration:

  • Platform: BBC Micro
  • Architecture: 6502
  • Configuration: none.cfg (generic 6502)
  • Library: none.lib
  • Target: bbc (-t bbc flag)
